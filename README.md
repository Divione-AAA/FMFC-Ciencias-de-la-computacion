# FMFC — Ciencias de la Computación 🚀

🎓 **Banner:** 🚀👩‍💻👨‍💻✨

Clases, ejercicios y proyectos personales organizados por lenguaje y área. Este repositorio reúne trabajos realizados durante la carrera: prácticas, resolución de problemas competitivos, estructuras de datos, proyectos web y experimentos en varios lenguajes.

**Resumen visual**: 🧠💻🎯

**Tabla de contenidos**
- **`C/`** — Proyectos y ejercicios en C
- **`C++/`** — Algoritmos, estructuras y competiciones en C++
- **`JAVA/`** — Proyectos y prácticas en Java (interfaces, manejo de excepciones)
- **`PYTHON/`** — Scripts, pruebas y proyectos rápidos en Python
- **`HTML/`, `CSS/`, `JAVASCRIPT/`, `TAILWIND/`** — Prototipos front-end y ejercicios web
- **`DJANGO/`**, **`GRAPHQL/`** — Backend y APIs
- **`ESTRUCTURA DE DATOS/`** — Implementaciones en Java y material de teoría
- **`ICPC2/`** — Carpetas de competiciones y algoritmos (dijkstra, kruskal, backtracking...)
- **`ENSAMBLADOR/`, `DART/`, `SQL/`** — Otros lenguajes y prácticas
- **`REACT.JS/`** — Practicas, hooks, props y otros conceptos

---

**¿Qué encontrarás aquí?**

- **Proyectos por carpeta**: cada carpeta contiene ejercicios o mini-proyectos relacionados con el lenguaje o la materia.
- **Código educativo**: soluciones a problemas, pruebas de concepto y apuntes.
- **Enfoque práctico**: compilables y editables en tu máquina (varios CMake/C++ y scripts).

---

**Descripción por carpeta (resumen)**

- **`C/`** 🟦
	- Contiene: ejercicios en C, prácticas de laboratorio y soluciones (p.ej. `main.cpp`, `p1.c`, `p11.c`).
	- Habilidades: programación procedimental, manejo de memoria, punteros y archivos.
	- Lenguajes / herramientas: `C`, `gcc`, `Make`/`CMake`.
	- Emoji: 🔧🧩

- **`C++/`** 🟦
	- Contiene: `auction.cpp`, ejercicios para prácticas y estructuras de datos.
	- Habilidades: programación orientada a objetos, STL, algoritmos y optimización.
	- Paradigmas: OOP, genérico (templates) y algorítmico.
	- Emoji: 🧠⚙️

- **`JAVA/`** ☕
	- Contiene múltiples ejercicios y proyectos (interfaces gráficas, estructuras, manejo de excepciones, proyectos Maven).
	- Habilidades: OOP avanzado, estructuras de datos en Java, GUI básica, uso de `maven`.
	- Emoji: ☕🧩

- **`PYTHON/`** 🐍
	- Contiene: scripts, pruebas y pequeños proyectos. Ideal para prototipar ideas o automatizar tareas.
	- Habilidades: scripting, manipulación de archivos, librerías comunes (si se usan), programación funcional y orientada a objetos.
	- Emoji: 🐍📜

- **`HTML/`, `CSS/`, `JAVASCRIPT/`, `TAILWIND/`** 🎨
	- Contienen: ejemplos de interfaces, tablas, formularios y demostraciones de layouts (`GRID`, `ESPACIADO`, `DATALIST`).
	- Habilidades: maquetación, responsive design, DOM, estilos con Tailwind/CSS puro.
	- Emoji: 🎨🖥️

- **`DJANGO/``**, **`GRAPHQL/`** 🗄️
	- Contienen: proyectos backend y experimentos con APIs (Django, GraphQL). Ideal para aprender arquitecturas cliente/servidor.
	- Habilidades: REST/GraphQL, ORM, rutas, autenticación básica.
	- Emoji: 🗄️🔐

- **`ESTRUCTURA DE DATOS/`** 🧾
	- Implementaciones en Java: árboles, pilas, colas, tablas hash, AVL, B-tree y más.
	- Habilidades: diseño de estructuras, complejidad temporal/espacial, implementación desde cero.
	- Emoji: 🌳📊

- **`ICPC2/`** 🏆
	- Carpetas por técnica: `dijkstra/`, `kruskal/`, `backtracking/`, `greedy/`, etc.
	- Habilidades: resolución de problemas, análisis de casos límite, programación competitiva.
	- Emoji: 🏆⚡

- **`ENSAMBLADOR/`** ⚙️
	- Ejercicios bajos niveles y prácticas de arquitectura.
	- Habilidades: comprensión de instrucciones, registros, memoria y optimización a bajo nivel.


- **`DART/`** y otros 🧪
	- Pequeños experimentos con Dart y otras tecnologías.

- **`TYPESCRIPT/`** 🟪
	- Ejercicios y proyectos escritos en TypeScript, orientados a la práctica de tipado estático sobre JavaScript y desarrollo moderno de aplicaciones web.
	- Habilidades: tipado estático, desarrollo web, integración con frameworks modernos, buenas prácticas de JavaScript avanzado.
	
- **`SQL/`** 🗃️
	- Consultas, esquemas y prácticas para modelado relacional.

**Habilidades principales que desarrollé** ✨
- **Lenguajes:** `C`, `C++`, `Java`, `Python`, `JavaScript`, `HTML/CSS`, `SQL`, `Dart`, `Assembly`.
- **Paradigmas:** programación procedimental, orientada a objetos (OOP), programación funcional (cuando aplica), y programación declarativa en SQL.
- **Áreas:** estructuras de datos, algoritmos, programación competitiva, desarrollo web (front + backend), bases de datos, compilación y herramientas (CMake, Make).
- **Herramientas:** `git`, `g++/gcc`, `javac/java`, `python`, entornos IDE (VS Code, IntelliJ), `maven`, `pip`, `virtualenv`.

**Consejos rápidos para ejecutar/compilar**
- C/C++ (PowerShell):

```powershell
g++ -std=c++17 -O2 -Wall C/main.cpp -o bin/main.exe; .\bin\main.exe
```

- Java:

```powershell
javac -d out Java/SomeFile.java; java -cp out paquete.Principal
```

- Python:

```powershell
python PYTHON/script.py
```

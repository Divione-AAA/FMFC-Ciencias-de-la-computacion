# 📚 Estructura de Datos 2 - Grafos

Este repositorio contiene implementaciones y experimentos relacionados con grafos en Java. A continuación se describe el propósito, estructura y aprendizajes de cada archivo:

---

## 🗂️ Grafo.java
**Descripción:**
- Define la interfaz o clase base para representar un grafo.
- Sienta las bases para las implementaciones específicas de grafos.

**Aprendizajes:**
- Comprensión de la abstracción de grafos.
- Importancia de definir métodos comunes para diferentes representaciones.

---

## 🧩 GrafoListaAdyacencia.java
**Descripción:**
- Implementa un grafo usando listas de adyacencia.
- Cada vértice tiene una lista de sus vecinos.

**Aprendizajes:**
- Eficiencia en espacio para grafos dispersos.
- Manipulación de listas enlazadas o estructuras similares.
- Cómo agregar y buscar conexiones rápidamente.

---

## 🟦 GrafoMatrizAdyacencia.java
**Descripción:**
- Implementa un grafo usando una matriz de adyacencia.
- Utiliza una matriz bidimensional para representar conexiones entre vértices.

**Aprendizajes:**
- Facilidad para verificar la existencia de aristas.
- Consumo de espacio en grafos densos.
- Operaciones rápidas para grafos pequeños o densos.

---

## 🔍 smatching.java
**Descripción:**
- Archivo relacionado con algoritmos de matching o emparejamiento en grafos.
- Puede contener implementaciones de algoritmos para encontrar emparejamientos máximos o perfectos.

**Aprendizajes:**
- Aplicación de algoritmos avanzados en grafos.
- Resolución de problemas de emparejamiento.
- Análisis de eficiencia y casos de uso.

---

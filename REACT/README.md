# FMFC Ciencias de la Computación - REACT

Este repositorio contiene ejemplos y ejercicios prácticos sobre conceptos fundamentales de React, orientados al aprendizaje de Ciencias de la Computación. El objetivo es facilitar el aprendizaje de los principales temas de React mediante archivos independientes y bien organizados.

## Estructura de archivos
- **componentesfuncionales.tsx**: Ejemplos de componentes funcionales en React.
- **hooks.tsx**: Uso de hooks como useState, useEffect y personalizados.
- **props.tsx**: Ejemplos de paso y manejo de props entre componentes.
- **renderconlistas.tsx**: Renderizado dinámico de listas y arrays.
- **renderizado condicional.tsx**: Técnicas para renderizar componentes de forma condicional.
- **reutilizables.tsx**: Componentes reutilizables y patrones de composición.

## Requisitos
- Node.js y npm instalados
- Editor de código (recomendado: Visual Studio Code)
